 
  ✨ StarSync v1.0 - Smart File Sorting

**StarSync** is an aesthetic and powerful Python-based utility designed to bring order to your folders. It’s time to forget the chaos in your "Downloads"! Just drag a folder into the app and let the magic begin.

## Key Features
- **Drag-and-Drop**: Simply drop a folder into the program window.
- **Smart Themes**: 12 color schemes (Forest, Abyss, Amethyst, Wine, etc.) with support for dark and light modes.
- **Multilingual**: RU / EN / ES / ZH.
- **Flexible Logic**: Sort by file format or date (day/month/year).
- **Safety First**: Automatic filename conflict resolution (prevents overwriting).
- **Logging**: Full real-time activity history.

## Tech Stack
- **Python 3**
- **Tkinter** (Custom Canvas UI)
- **TkinterDnD2** (for Drag-and-Drop)
- **Threading** (smooth, lag-free performance)

## Setup & Launch
###For Users (Windows):
1. Navigate to the Releases section https://github.com/GalLogicx/StarSync.
2. Download the StarSync_v1.0_by_GalLogicx.zip archive.
3. Extract and run StarSync.exe. No installation required!

###For Developers:
1. Clone the repository:' git clone https://github.com/GalLogicx/StarSync'
2. Install dependencies: 'pip install -r requirements.txt'
3. Run the app: python main.py

 ✨ StarSync v1.0 - Умная Сортировка Файлов


**StarSync** - это эстетичная и мощная утилита на Python для наведения порядка в папках. Время забыть о хаосе в "Загрузках"! Можно перетащить папку в приложение, и магия начнется.

## Основные фишки
- **Drag-and-Drop:** Просто брось папку в окно программы.
- **Умные темы:** 12 цветовых схем (Лес, Бездна, Аметист, Вино и др.) с поддержкой темного и светлого режимов.
- **Мультиязычность:** RU / EN / ES / ZH.
- **Гибкая логика:** Сортировка по форматам, датам (день/месяц/год).
- **Безопасность:** Автоматическое разрешение конфликтов имен (защита от перезаписи).
- **Логирование:** Полная история действий в реальном времени.

## Технологии
- **Python 3**
- **Tkinter** (Custom Canvas UI)
- **TkinterDnD2** (для Drag-and-Drop)
- **Threading** (плавная работа без зависаний)

## Установка

### Для пользователей (Windows):

2. Скачайте архив "StarSync_v1.0_by_GalLogicx.zip".
3. Распакуйте и запустите StarSync.exe. 
**Установка не требуется!**

### Для разработчиков:
1. Клонируйте репозиторий: "git clone https://github.com/GalLogicx/StarSync"
